==========
easyhelium
==========


Track your recent Helium wallet activities. Easily.


Usage
===========

.. code-block:: bash

   usage: easyhelium.py [-h] [--last {24h,48h,7d,14d,30d}] [--currency {eur,usd}] wallet_id
